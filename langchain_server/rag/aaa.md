# 日本語ドキュメントの表の解釈

質問:
7月入社の社員の有給休暇日数は？


回答

qwen3:4b

💡 回答:
7月入社の社員の有給休暇日数は、**15日**です。  

**根拠**:  
- [就業規則_202509.pdf p.5]の表「入社月 有給休暇日数」によると、**7月入社の場合は15日**が定められています。  
- ただし、勤続年数に対する有給休暇日数の計算では、前年度の出勤率（8割以上）が条件ですが、この表は入社月ごとの初期付与日数を示しており、**入社月の影響で15日が定められています**。  
- したがって、7月入社の社員は**15日**の有給休暇を取得できます。
